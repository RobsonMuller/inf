Description

A software library for retrieving email messages from POP3 mailserver.

Works with programming and scripting languages supporting .NET or COM

Supports secure email(SSL), complete and partial message retrieval, email (MIME) attachments, mailbox status check and reset

Installation

1. download OSPOP3.zip

2. unzip downloaded file to a temporary directory

3. copy OSPOP3_Plus.dll to a root directory of your .NET project

* if you are using the component from VB6 or other COM language copy both OSPOP3_Plus.dll and OSPOP3_Plus.tlb to a system directory, then register OSPOP3_Plus.tlb
